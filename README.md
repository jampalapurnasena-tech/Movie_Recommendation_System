# Movie Recommendation System

This project recommends movies based on similarity using Machine Learning techniques.

## Features

* Recommends similar movies
* Content-based filtering
* Cosine similarity algorithm

## Technologies Used

* Python
* Pandas
* Scikit-learn
* Google Colab

## Dataset

TMDB 5000 Movie Dataset

## Example

Input: Avatar

Recommended Movies:

* Guardians of the Galaxy
* Star Trek
* John Carter
